# Свадебное приглашение

## Запуск
Откройте `index.html` в браузере.

## Публикация на Netlify
1. Создайте репозиторий на GitHub.
2. Загрузите туда все файлы проекта.
3. В Netlify выберите Add new site → Import an existing project → GitHub.
4. Build command оставьте пустым.
5. Publish directory: `.`
6. Нажмите Deploy.

## Ответы формы
После публикации ответы будут в Netlify: Site dashboard → Forms → rsvp.
Их можно скачать CSV и открыть в Excel.

Важно: локально форма не отправляется, она работает после публикации на Netlify.
